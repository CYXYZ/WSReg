# WSReg

Official implementation of **WSReg**, a weakly supervised semantic consistency framework for 2D/3D registration, accepted at **MICCAI 2025**.

The code is currently being cleaned and organized. We plan to release it soon. Please stay tuned!

For updates, please check back here or contact us if you have any questions.
